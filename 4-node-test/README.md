### Testing

#### Highlight

* [jest](https://facebook.github.io/jest/)
* async testing
* [supertest](https://github.com/visionmedia/supertest)
* [rewire](https://github.com/jhnns/rewire)

#### Usage

```bash
$ yarn  # install the dependencies
$ yarn jest # run jest
$ yarn jest:w  # run jest with watching mode
```
